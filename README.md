# Sistem Booking Alat Tani Berbasis Blockchain

## Deskripsi
Sistem ini digunakan untuk melakukan booking alat tani desa seperti traktor komunitas menggunakan smart contract Solidity tanpa sistem pembayaran.

## Aktor
- Admin Desa
- Warga

## Fitur
- Menambahkan alat tani
- Booking alat
- Status alat: Kosong / Terpesan

## Smart Contract
contracts/BookingAlatTani.sol

## Teknologi
- Solidity
- Remix IDE
- Ethereum Virtual Machine
- GitHub
